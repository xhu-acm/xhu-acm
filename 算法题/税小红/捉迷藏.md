[捉迷藏](https://www.matiji.net/exam/brushquestion/10/3181/1DC60EA6DF83A333301CFFE1407FBA59)


自己做题时的思考：
每个位置可以作为初始放置的位置就三种情况：重头到尾未被询问；左边位置先被询问，自己后被询问；右边位置先被询问，自己后被询问。遍历每个位置，统计结果。


题解：记录每一个位置最晚的询问和最早的询问时间。之后遍历情况：
1.如果改位置没有被询问过，则ans++
2.如果该位置最早的询问时间比左右两侧的询问时间长，则可以移动，按照情况分别ans+1即可

最后的代码：
#include<bits/stdc++.h>
using namespace std;
#define INF 0x3f3f3f3f 
const int N=100001;
int a[N],f[N],l[N],ans;
int n,m;
int main()
{
	cin>>n>>m;
	memset(l,-1,sizeof(l));
	memset(f,INF,sizeof(f));
	for(int i=1;i<=m;i++)
	{
		cin>>a[i];
		f[a[i]]=min(f[a[i]],i);
		l[a[i]]=max(l[a[i]],i);
	}
	for(int i=1;i<=n;i++)
	{
		if(i>1&&f[i]>l[i-1])ans++;
		if(i<n&&f[i]>l[i+1])ans++;
		if(l[i]==-1)ans++;
	}
	cout<<ans;
	return 0;
}
