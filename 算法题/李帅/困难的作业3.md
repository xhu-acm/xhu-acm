	思考：题目需要输入一串字符串并统计其中每个字母出现的次数。
	由于cin不能直接输入空格所以我选择使用cin.getline()输入。
	对于字母个数的统计可以使用if语句实现。
	但如果对每个字母都单独写一个if就太麻烦了。
	我通过定义并初始化变量sd='a'，并每次创建for(i)循环使从a到z的每个字母都被统计到。
	我在for(i)循环中嵌套for(j)循环遍历目标字符串中的每一个元素统计a+i字母出现的次数。


    题解（代码）：
    ```
	#include <iostream>
	#include <cstring>
	#include<string>
	#include<vector>
	using namespace std;
	int main()
	{
		string l;
		char sd = 'a';
		vector<int> a(26,0) ;
		int w;
		getline(cin, l);
		for (char i = 0; i < 26; i++)
		{
			for (int j = 0; j < l.size(); j++)
			{
				if (l[j] == sd + i)
				{
					a[i]++;
				}
			}
			cout << a[i] << " ";
		}
		return 0;
	}
    ```
