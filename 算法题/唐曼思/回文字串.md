[回文字串](https://www.luogu.com.cn/problem/P1435)
自己做题时的思考：
  本题需要求出将给定字符串变成回文词所需插入的最少字符数。如果一个区间两边的字符相同，则只要让这
  个字符中间的区间是回文字串就行。所以我们需要求出不同长度的字符区间变成回文字串的最小插入字符数。
题解：
  首先定义一个dp数组a[i][j],表示在i到j的区间中形成回文字串所需插入的最小字符数；
  如果s[i]==s[j]那么就不需要额外插入字符，则a[i][j]=a[i+1][j-1];
  如果s[i]!=s[j]时，要让i到j这个区间为回文字串，让区间首尾相等有两种情况
    1、在s[j]后插入s[i];
    2、在s[i]前插入s[j];
  a[i][j]==min(a[i+1][j],a[i][j-1])+1;
最后代码：
#include<iostream>
using namespace std;
const int N = 1e3;
int a[N][N] = { 0 };
int main()
{
	string s;
	cin >> s;
	int n = s.size();
	s = ' ' + s;//让索引从1开始
	for (int len = 2; len <= n; len++)//区间长度
	{
		for (int i =1; i + len - 1 <= n; i++)//i<=j;
		{
			int j = i + len - 1;
			if (s[i] == s[j])a[i][j] = a[i +1][j - 1];
			else
			{
				a[i][j] = min(a[i + 1][j], a[i][j - 1])+1;
			}
		}
	}
	cout << a[1][n];
	return 0;
}
